### Интегрировано: ReactJS, ESLint (синтаксис и стиль кода), пакетный менеджер yarn.
### Планируется: Webpack, Prettier.

# Инструкция (кратко)

### Запуск проекта

`npm start` или `yarn start`

### Скачивание пакетов проекта

`npm i` или `yarn`

### Сборка проекта

`npm run build` или `yarn build`
